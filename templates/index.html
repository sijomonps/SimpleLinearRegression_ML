What is Throughput?

Throughput is the number of requests a server processes per second (req/s). A higher throughput means the server is under heavier load, which typically increases response time.

Unit: requests / second
⏱

What is the Output?

The model predicts the Response Time — the time the server takes to reply to a single request. Lower values indicate better performance.

Unit: seconds
